# Logs Analysis Project of Udacity Full Stack Nanodegree Program

## How to run
1. git clone this repository
2. cd <to the directory where the Vagrantfile is>:
```sh
vagrant up
vagrant ssh
```
4. inside vagrant:
```sh
cd /vagrant/logs_project/
python logs_analysis.py
```
